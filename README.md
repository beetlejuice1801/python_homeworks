Проект представляет собой веб-приложение для управления библиотекой книг, построенное на FastAPI. Приложение включает как веб-интерфейс (HTML страницы), так и полноценное REST API для работы с книгами.

    Веб-интерфейс: HTML страницы с навигацией и адаптивным дизайном

    Полноценное API: CRUD операции для работы с книгами

    Автодокументация: Интерактивная документация API (Swagger/ReDoc)

    Валидация данных: Строгая типизация и валидация с помощью Pydantic

    Шаблонизация: Рендеринг HTML через Jinja2
